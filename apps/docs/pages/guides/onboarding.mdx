# Developer Onboarding

Welcome to the **Lattice** engineering team! We're excited to have you helping us build the best event discovery platform for Barcelona.

## Getting Started

Follow these steps to set up your development environment:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/lattice-product-team/app_lattice_project.git
    cd app_lattice_project
    ```
2.  **Read the Workflow**: Familiarize yourself with our [Monorepo Workflow](../engineering/monorepo-workflow.md).
3.  **Local Setup**:
    - Refer to our fast [Getting Started Guide](./getting-started.md) to launch the monorepo services in 5 minutes.
    - For a detailed step-by-step setup of the PostGIS database, Redis caches, and environment variables, read our comprehensive [Installation Manual](./installation.md).
    - If you hit any local network binding, port collision, or package compilation errors, consult the [Troubleshooting Guide](./troubleshooting.md).

## Your First Week

- **Day 1**: Environment setup and repository exploration.
- **Day 2**: Read the [System Overview](../architecture/system-overview.md) and [Product Vision](../product/vision.md).
- **Day 3**: Set up your IDE with ESLint and Prettier. Run the project locally.
- **Day 4**: Pick a small "Good First Issue" from the backlog.
- **Day 5**: Submit your first Pull Request!
