This guide explains how to use the Lattice Admin Dashboard to manage the platform and curate content.

## 1. Content Management

### Managing Events
- **Creating an Event**: Click on the "Create Event" button. You will need to provide a name, dates, and a primary color for branding.
- **Setting Boundaries**: Use the map tool to draw the polygonal boundary of the event. This determines where the custom navigation network will be active.

### Managing POIs (Points of Interest)
- **Adding POIs**: You can add POIs (like WCs, medical points, or stages) by clicking on the map.
- **Status Updates**: You can change a POI status to `Maintenance` or `Closed` in real-time, which will be immediately reflected in the Mobile App.

---

## 2. Telemetry & Monitoring

### Real-time Occupancy
The dashboard displays the current occupancy of key areas based on ticket scans and telemetry logs. 
- **Crowd Levels**: `Low`, `Moderate`, `High`, `Blocked`.
- **Alerts**: If an area reaches "Blocked" status, an alert will be sent to operational staff.

### System Health
Check the "System" tab to monitor:
- API response times.
- Database load.
- Active user counts.

---

## 3. Accessibility Curation

Lattice prides itself on being inclusive. Administrators are responsible for:
- Marking POIs as `Wheelchair Accessible`.
- Identifying path segments with `Stairs` or `Steep Slopes` in the navigation network editor.
