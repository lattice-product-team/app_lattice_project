# User Manual

Esta guía proporciona instrucciones detalladas sobre cómo utilizar la aplicación Lattice para maximizar la experiencia en el evento.

## 1. Primeros Pasos
Para comenzar, descarga la aplicación y crea un perfil. Puedes sincronizar tus entradas físicas escaneando el código QR para tener todo en tu "Wallet" digital.

## 2. Navegación y Descubrimiento
- **Búsqueda de Servicios**: Utiliza la pestaña de descubrimiento para encontrar restauración, servicios médicos o tribunas.
- **Mapa Interactivo**: Pulsa sobre los marcadores para ver el estado de congestión en tiempo real.
- **Navegación AR**: Activa el modo "Brújula AR" para recibir indicaciones visuales sobre la imagen de tu cámara, ideal para orientarse entre la multitud.

## 3. Funciones Avanzadas
- **Gestión de Grupos**: Crea un grupo con tus amigos para ver su ubicación en tiempo real sobre el mapa y evitar perderos.
- **Preferencias de Accesibilidad**: Configura tu perfil (silla de ruedas, movilidad reducida, etc.) para que las rutas calculadas eviten escaleras o pendientes pronunciadas.
