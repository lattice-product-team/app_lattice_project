# F - Functional & Commercial Presentation

This section outlines the project's vision, functionality, and market potential.

## 1. Project Definition
Lattice is a smart discovery engine for urban events, bridging the gap between local activities and user navigation.

### Objectives
- Simplify event discovery in high-density urban areas.
- Provide real-time navigation aids (interactive maps & AR).
- Enable seamless ticket synchronization and profile management.

### Scope
- **Target Audience**: Urban explorers, event-goers, and tourists.
- **Platforms**: iOS/Android (via React Native) and Web (Admin).

## 2. Core Functionalities
- **Event Discovery**: Dynamic filtering based on category, time, and location.
- **Interactive Maps**: Custom markers with real-time status.
- **User Journeys**: [Detailed User Flows](./user-journeys.md)

## 3. Market Study
- **Competitors**: Google Maps, Eventbrite, specialized local apps.
- **Differentiator**: High-performance "Neon" UI and dedicated navigation tools (AR compass, HUD).

## 4. Financial Plan
- **Initial Costs**: Infrastructure, API subscriptions, development time.
- **Maintenance**: Hosting, data ingestion services.
- **Infrastructure**: AWS/GCP, Docker-based deployment.
