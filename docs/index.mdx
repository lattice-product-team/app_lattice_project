# Introducción a App Lattice

Bienvenido a la documentación oficial de **App Lattice**. Este proyecto es una plataforma fullstack diseñada para escalar, utilizando tecnologías modernas como **Next.js**, **Expo**, **Express** y **PostgreSQL**.

## Estructura del Proyecto

Nuestra arquitectura está organizada como un monorepo administrado por **pnpm** y **Turbo**:

- **`apps/mobile`**: Aplicación móvil multiplataforma construida con Expo.
- **`apps/admin-web`**: Panel de administración web construido con Next.js.
- **`apps/server`**: API robusta construida con Express y Prisma.
- **`packages/`**: Librerías compartidas y configuración.

## Tecnologías Principales

- **Frontend**: Next.js 16, React 19, Tailwind CSS 4.
- **Mobile**: Expo 54, React Native 0.81.
- **Backend**: Node.js, Express, PostgreSQL.
- **Tooling**: pnpm, Turbo, TypeScript.
