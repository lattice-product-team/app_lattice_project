# Activity Diagrams

The Lattice activity flowcharts have been split into two interactive, zoomable, and pan-enabled versions to maximize detail and readability:

*   **[Mobile Activity Diagram](file:///home/nildiaz/app_lattice_project/docs/architecture/activity-diagram-mobile.md)**: Models the end-to-end client lifecycle of the mobile application (Expo Router launch synchronization, onboarding flows, dual-canvas sliding layout, expandible category search island, multi-profile route planning, active turn-by-turn guidance, and device-tilt adaptive AR overlay HUD).
*   **[Web-Admin Activity Diagram](file:///home/nildiaz/app_lattice_project/docs/architecture/activity-diagram-admin.md)**: Models the operational center workflow of the Next.js administrative dashboard (secure middleware checks, Server Action login, viewport bounds orchestration, live WebSocket updates, Crowd Radar telemetry polling, and full-screen coordinate/boundary drawing overlays).
