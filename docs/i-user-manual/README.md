# I - User Manual

This guide provides instructions on how to use the Lattice application effectively.

## 1. Getting Started
To begin using Lattice, download the app and create an account. You can sync your tickets and preferences across devices.

## 2. Navigation & Discovery
- **Finding Events**: Use the discovery tab to browse events by category (Music, Sports, Tech).
- **Interactive Map**: Tap on markers to see detailed information about POIs and ongoing events.
- **AR Navigation**: Use the "Compass" mode for real-time directional guidance in complex areas.

## 3. Advanced Features
- **Profile Management**: Update your interests to receive personalized recommendations.
- **Localization**: The app supports multiple languages. [Read more](./localization.md)
- **Accessibility**: We follow strict design standards to ensure usability for everyone. [Design Standards](./design-standards.md)
