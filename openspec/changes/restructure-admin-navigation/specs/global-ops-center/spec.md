## MODIFIED Requirements

### Requirement: Global Operations Dashboard
The system SHALL provide a centralized "Global Asset Map" as the root page of the Admin Web application, displaying all assets (Events, POIs) and real-time telemetry instead of aggregated metrics.

#### Scenario: Global Map Visualization
- **WHEN** the user visits the root path `/`
- **THEN** the system MUST render the full-screen Global Map with active asset markers.
