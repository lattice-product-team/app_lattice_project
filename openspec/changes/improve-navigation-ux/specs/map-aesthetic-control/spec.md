## ADDED Requirements

### Requirement: Puntero de Usuario con Brújula
El sistema SHALL mostrar un indicador de orientación (heading) en el puntero de usuario.

#### Scenario: Rotación del puntero
- **WHEN** el usuario cambia la orientación del dispositivo físicamente
- **THEN** el icono del usuario en el mapa rota en tiempo real para reflejar la dirección hacia la que apunta el dispositivo
